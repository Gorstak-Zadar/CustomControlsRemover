# 🧹 CustomControlsRemover

> **Malicious UI Control Removal** - Detects and removes malicious custom window controls.

---

## 📋 Overview

CustomControlsRemover detects and removes malicious custom controls that may be injected into applications or the desktop environment to trick users or capture input.

---

## 🎯 What It Does

- 🔍 **Control Detection** - Find malicious UI controls
- 🗑️ **Safe Removal** - Remove suspicious controls
- 🛡️ **Injection Prevention** - Block control injection
- 👁️ **Visual Monitoring** - UI element monitoring

---

## 🚀 Usage

```powershell
# Remove malicious controls
.\CustomControlsRemover.ps1
```
